# Mythic Bastionland Character Tracker

A browser-based character sheet for [Mythic Bastionland](https://www.bastionland.com/) by Chris McDowall. No server, no install, no account. Open the page and start playing.

All data is stored in your browser's local storage. Export and import characters as JSON files to back up your data or share characters between devices.

## Features

**Character Management**
- Create characters from 30+ knight archetypes or build a fully custom knight
- Track Virtues (VIG, CLA, SPI), Guard Dice, Glory, Rank, Age, and Season
- Manage multiple characters with instant switching
- Export/import individual characters as JSON

**Combat**
- Dice pool builder with weapon selection, shield die, charging, and bonus dice
- Feat tracking (Smite, Focus, Deny) with save rolls and fatigue states
- Damage calculator with automatic armour subtraction, wound detection, and mortal wound alerts
- Scar system triggered on Guard break (d12 table with Doom tracking)
- Mounted combat with trample dice and dismount handling
- Gambits reference panel

**Equipment**
- Searchable database of all standard weapons, armour, and beasts
- Custom item creation with dice, tags, and armour scores
- Equipped/carried/inventory management
- Shield trapping and weapon impairment toggles

**Tracking**
- Conditions panel with auto-derived states (Exhausted, Exposed, Impaired, Fatigued)
- Remedy tracking (Sustenance, Stimulant, Sacrament)
- Mount and squire companion sheets with full stats
- Contacts list with descriptions and favors
- Season log for downtime pursuits
- Scar history with effects and dates
- Auto-journaling of significant actions with undo/restore

**Help System**
- Contextual "?" tooltips throughout the app explaining game mechanics
- Covers virtues, combat, feats, gambits, equipment, conditions, and more

**Backup**
- Visual backup status indicator (green/orange/red)
- One-click full backup of all characters
- Import supports both single characters and full backup files

## Usage

Visit [qtwre.github.io/mbCharSheet](https://qtwre.github.io/mbCharSheet/) or download `index.html` and open it directly in your browser.

Everything runs client-side. No data leaves your machine.
